# 图片水印参数预览设计

## 背景
现有“批量图片文字水印”工具支持调整文字、字号、透明度、旋转角度、横纵向间距，但用户只能依赖经验填写参数，缺少可视化反馈。

## 目标
在不改动 Tauri 后端命令与参数结构的前提下，为批量图片水印功能新增一个前端实时预览区域，帮助用户在提交前快速观察参数组合效果。

## 方案
- 在 `BatchImageWatermarkTool` 中新增“参数预览”区块。
- 预览使用固定尺寸的透明棋盘格背景，不依赖用户真实图片。
- 在预览区域内以前端 DOM/CSS 平铺多组水印文字，联动以下参数：
  - 水印文字
  - 字号
  - 透明度
  - 旋转角度
  - 横向间距
  - 纵向间距
- 预览仅作为调参参考，不追求与最终每张图片像素级一致。

## 边界
- 不新增依赖。
- 不改动 Tauri 命令、后端 Rust 实现、invoke payload 结构。
- 不引入真实图片选择或额外预览文件生成逻辑。

## 测试策略
- 为批量图片水印工具补充测试，验证预览区默认渲染。
- 验证修改参数后预览文本与关键样式会更新。
- 验证不影响现有提交按钮禁用、进度展示与 invoke 提交流程。
